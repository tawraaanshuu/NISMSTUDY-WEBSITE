🚨 Troubleshooting:

  • If you get "ERR_FILE_NOT_FOUND": Make sure you're using the full URL with http://localhost:5000/
  • If you get "Connection refused": The server is not running. Start it with npm start
  • If you get "Port already in use": Kill existing processes or use npm run demo
  • If links work but show errors: Check browser console (F12) for JavaScript errors
